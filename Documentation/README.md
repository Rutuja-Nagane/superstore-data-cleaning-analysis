# Documentation

This folder contains project documentation related to the data cleaning and preprocessing phase of the Superstore Sales dataset.

## Included File
- `Data_Cleaning_Steps.pdf`

## Documentation Covers
- Issues identified in raw dataset
- Data cleaning techniques performed
- Date format standardization
- Duplicate removal
- New analytical columns created
- Importance of data preprocessing

The cleaned dataset prepared in this phase will be used for further SQL analysis and Power BI dashboard development.
